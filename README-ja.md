# Space Panicco Archive

PC-98用アクションゲーム  
**Space Panicco（すぺーす ぱにっ娘）** の歴史的アーカイブです。

初公開は **1994年** です。

このリポジトリは、**オリジナル作者の一人** によって作成されました。

このリポジトリは、当時の配布ファイル、ドキュメント、
ディスクイメージ、および関連する実行環境を、歴史保存およびアーカイブ目的で保管しています。

## ゲームについて

Space Panicco は、**Space Panic に着想を得たプラットフォームアクションゲーム** です。

プレイヤーは穴を掘り、敵を落とし、攻撃を避けながら各ラウンドを攻略していきます。  
本作は **NEC PC-9801 シリーズ** 向けに開発され、**MS-DOS** 上で動作します。

## リポジトリ構成

- `original/` — 当時のオリジナル配布アーカイブファイルおよびディスクイメージ
- `extracted/` — オリジナルアーカイブから展開した内容
- `docs/` — ドキュメント、マニュアル、ライセンス注記、および関連ノート
- `screenshots/` — ゲームのスクリーンショット
- `environment/` — エミュレータおよび実行環境ファイルと関連ノート
- `analysis/` — 保存目的の解析およびリバースエンジニアリングノート

---

## オリジナル配布物

`original/` ディレクトリには、**Space Panicco** の当時のオリジナル配布ファイルが保存されています。

これらのファイルは、歴史保存およびアーカイブ目的で保管されています。

内容には以下が含まれます。

- オリジナルのアーカイブパッケージ
- ディスクイメージ

展開済みファイルは、`extracted/` ディレクトリに分けて保管されています。

解析、改変、再構成した資料は、これらの外側に配置するべきです。

## 起動方法

1. PC-98 エミュレータ環境を用意します。
2. 保存された DOS イメージ、または同梱の FreeDOS(98) ベース起動ディスクのいずれかで起動します。
3. ディスクイメージをマウントするか、展開済みゲームファイルを開きます。
4. メイン実行ファイルを起動します。

詳細は以下を参照してください。

- [Running the Game](docs/Running-the-Game.md)
- [Running the Game (Japanese)](docs/Running-the-Game_ja.md)
