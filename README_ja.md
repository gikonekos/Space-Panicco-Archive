## ゲーム概要

Space Panicco は  
アーケードゲーム **Space Panic** に着想を得た
アクションゲームです。

タイトルは *Space Panic* のもじりで、  
主人公の名前は **ぱにっ娘（Panicco）** です。

プレイヤーは主人公 **ぱにっ娘** を操作し、  
床を掘って穴を作り、エイリアンを落としたり  
埋めたりして倒します。

本作では次のような要素が追加されています：

- 敵ごとに必要な落下高さが異なる
- 掘れない床を利用した **埋め殺し戦術**
- 敵同士の衝突や連鎖落下
- ステージデータ編集
- 設定ファイルによるゲーム調整
