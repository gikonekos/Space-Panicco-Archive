# すぺーす ぱにっ娘 アーカイブ

PC-98用アクションゲーム  
**すぺーす ぱにっ娘 (Space Panicco)** の歴史保存アーカイブです。

**1994年** に公開された作品です。

このリポジトリは、**当時の作者の一人** により作成されました。

ゲームの配布ファイル、ドキュメント、ディスクイメージ、および関連する実行環境を、
歴史資料として保存することを目的としています。

## ゲーム概要

すぺーす ぱにっ娘 は、**Space Panic** に影響を受けた
プラットフォームアクションゲームです。

穴を掘って敵を落とし、各ラウンドをクリアしていきます。
**NEC PC-9801 シリーズ** 向けに開発され、
**MS-DOS** 上で動作します。

## リポジトリ構成

- `original/` — 当時の配布アーカイブやディスクイメージ
- `extracted/` — 元アーカイブから展開した内容
- `docs/` — マニュアル、説明文書、ライセンスメモ、関連メモ
- `screenshots/` — ゲーム画面
- `environment/` — 実行環境やエミュレータに関するファイルとメモ
- `analysis/` — 保存・研究目的の解析資料や逆アセンブルメモ

---

## オリジナル配布物

`original/` ディレクトリには、**すぺーす ぱにっ娘** の当時の配布物を保存しています。

これらは歴史保存目的で保管されています。

内容には以下を含みます。

- 元の配布アーカイブ
- ディスクイメージ

展開済みファイルは `extracted/` に分けて保存しています。

解析資料、改変物、整理済み資料などは、原則としてこの外側に置いています。

## 実行方法

1. PC-98 エミュレータ環境を用意します。
2. 保存した DOS イメージ、または同梱の FreeDOS(98) 系起動ディスクを起動します。
3. ディスクイメージをマウントするか、展開済みファイルを開きます。
4. メイン実行ファイルを起動します。

詳細は以下を参照してください。

- [実行方法](docs/Running-the-Game_ja.md)
- [Running the Game](docs/Running-the-Game.md)
