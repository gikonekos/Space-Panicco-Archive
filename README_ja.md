## ゲーム概要

Space Panicco は  
アーケードゲーム **Space Panic** に着想を得た
アクションゲームです。

タイトルは *Space Panic* のもじりで、
主人公の名前は **ぱにっ娘（Panicco）** です。

プレイヤーは主人公 **ぱにっ娘** を操作し、
床を掘って穴を作り、エイリアンを落としたり
埋めたりして倒します。

本作では次のような要素が追加されています。

- 敵ごとに必要な落下高さが異なる
- 掘れない床を利用した **埋め殺し戦術**
- 敵同士の衝突や連鎖落下
- ステージデータ編集
- 設定ファイルによるゲーム調整

## 配布の歴史

本作は1994年に
日本のホビイスト向けソフト流通を通じて公開されました。

最初は **コミックマーケット** などのイベントで
フロッピーディスクと簡単な説明書を同封して頒布されました。

その後、オンライン配布として  
**Vector ソフトウェアライブラリ** に掲載されました。

現在は **GitHub（2026年）** 上で
PC-98同人ゲームの歴史保存プロジェクトとして公開されています。
