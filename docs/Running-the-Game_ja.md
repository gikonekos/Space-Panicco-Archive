## ゲームの実行方法

Space Panicco は **NEC PC-9801 シリーズ + MS-DOS** 環境向けに設計されたゲームです。

現在では **PC-98 エミュレータ** を使い、保存した元ディスクイメージ、または同梱の FreeDOS(98) 系起動ディスクから実行できます。

### 推奨エミュレータ

以下の PC-98 エミュレータで動作確認しています。

- Neko Project II
- Neko Project 21/W

### 動作確認環境

確認済み環境は以下の通りです。

- **Neko Project II x64** + **MS-DOS 3.30**
- **Neko Project 21/W x64** + **MS-DOS 6.20**
- **Neko Project II x64** + 同梱 **FreeDOS(98) 系起動ディスク**

---

### 方法1 — オリジナルのフロッピーディスクイメージ

このアーカイブに含まれている保存ディスクイメージを使用します。

```text
original/panic24.hdm
```

エミュレータでこのディスクをマウントし、DOS からゲームを起動してください。

実行ファイル:

```text
PANIC.EXE
```

---

### 方法2 — 同梱 FreeDOS(98) 系起動ディスク

**FreeDOS(98) 系 DOS 環境** を利用した起動可能フロッピーイメージも含まれています。

```text
environment/fd98_2hd_p24.img
```

このイメージは、現代の PC-98 エミュレータでゲームを動かしやすくするための補助起動環境です。

起動画面には、次のような構成要素の表示があります。

- FreeDOS(98) kernel
- FreeDOS XMS-Driver for 80286
- FreeCOM ver 0.85a_DBCS (PC-98)

同梱環境のライセンス関連メモについては、以下を参照してください。

- [FREEDOS98-LICENSE-NOTES_ja.md](FREEDOS98-LICENSE-NOTES_ja.md)

---

### 操作方法

移動

```text
8 : 上
2 : 下
4 : 左
6 : 右
```

アクション

```text
X   : 穴を掘る
Z   : 穴を埋める
ESC : ポーズ / 設定メニュー
```

テンキーの **2・4・6・8** はカーソルキーとして動作します。

---

### 注意

- 本ゲームは **PC-98 内蔵スピーカー（BEEP）** を使用します。
- ジョイスティックにも対応しています（必須ではありません）。
- ゲーム設定は以下のファイルで変更できます。

```text
PANIC.DEF
```

- ローカルでの動作確認では、`PANIC.EXE` 終了後にメッセージを表示するため、同梱起動環境側を便宜的に調整している場合があります。これは原作ゲーム本体そのものの変更ではありません。
