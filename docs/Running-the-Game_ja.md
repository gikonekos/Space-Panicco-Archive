## ゲームの実行方法

Space Panicco は **NEC PC-9801 シリーズ + MS-DOS**
環境向けに設計されたゲームです。

現在では **PC-98 エミュレータ**を使用することで
簡単にプレイできます。

---

### 推奨エミュレータ

以下の PC-98 エミュレータで動作確認しています。

- Neko Project II
- Neko Project 21/W

これらのエミュレータは
PC-98 ハードウェア環境を高い精度で再現します。

---

### 方法1 — フロッピーディスクイメージ

このアーカイブに含まれている
保存ディスクイメージを使用します。

original/panic24.hdm

エミュレータでこのディスクをマウントし、
DOS から以下を実行してください。

PANIC

実行ファイル

PANIC.EXE

---

### 方法2 — FreeDOS 起動ディスク

**FreeDOS(98)** を使用した
起動可能フロッピーイメージも含まれています。

environment/fd98_2hd_p24.img

このディスクイメージを起動すると
Space Panicco が自動的に開始します。

FreeDOS(98) 情報

https://bauxite.sakura.ne.jp/software/dos/freedos.htm

エミュレータ設定例

https://simk98.github.io/np21w/freedos98.html

---

### 操作方法

移動

8 : 上  
2 : 下  
4 : 左  
6 : 右  

アクション

X : 穴を掘る  
Z : 穴を埋める  
ESC : ポーズ / 設定メニュー  

テンキーの **2・4・6・8** は
カーソルキーとして動作します。

---

### 注意

- 本ゲームは **PC-98 内蔵スピーカー（BEEP）** を使用します
- ジョイスティックにも対応しています（必須ではありません）
- ゲーム設定は以下のファイルで変更できます

PANIC.DEF

---

### 互換性

動作確認環境

Neko Project II x64   + MS-DOS 3.30  
Neko Project 21/W x64 + MS-DOS 6.20  

また、付属の FreeDOS(98) 起動ディスクでも
ゲームを実行できます。

## 配布の歴史

本作は1994年に、日本のホビイスト向けソフト流通を通じて公開されました。

最初は **パソコン通信** により配布されました。

また、**コミックマーケット** などのイベントにおいて、  
フロッピーディスクと簡単な説明書を同封して頒布されました。

その後、オンライン配布として  
**Vector ソフトウェアライブラリ** に掲載されました。

現在は **GitHub（2026年）** 上で、  
PC-98同人ゲームの歴史保存プロジェクトとして公開されています。
