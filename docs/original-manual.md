# すぺーす ぱにっ娘 V2.4
## アクションゲーム版

**プログラム**：芸夢職人  
**音楽＆グラフィック**：基 建吉  
**日付**：1994-09-20

---

## 1. 概要

立体型エイリアン穴埋めゲームです。  
ギャルを操作して、エイリアンをすべて埋めて倒してください。

連鎖落としなどのテクニックも使えます。

---

## 2. ファイルの内容

| No. | ファイル名 | 内容 |
|---|---|---|
| 1 | `PANIC.EXE` | 実行ファイル |
| 2 | `PANIC.D32` | キャラデータ（32×32） |
| 3 | `MOJI.D16` | キャラデータ（16×16） |
| 4 | `PANIC.DEF` | 設定情報 |
| 5 | `VMAP***.DAT` | マップデータ（001-999） |
| 6 | `PANIC.BGM` | BGMデータ |
| 7 | `PANIC.EFS` | SOUNDデータ |
| 8 | `PANIC.DOC` | 本ファイル |

---

## 3. 実行方法

**すべてのファイルをカレントディレクトリに置いてください。**

```text
PANIC[RET]
```

タイトル画面が表示されたら、**Zキー** または **ジョイスティックのトリガB** でゲーム開始です。

---

## 4. 遊び方

あなたは **ぱにっ娘** を操作して、エイリアンをすべて倒します。

### エイリアンの倒し方

**敵エイリアンを穴に埋めて落としてください。**

各エイリアンは、必要階数以上落とさないと倒せません。

| エイリアン | 必要落下階数 |
|---|---:|
| `SLIMER` | 1階以上 |
| `EBYRINS` | 2階以上 |
| `ZAUROID` | 3階以上 |

ただし、落としたエイリアンがその真下のエイリアンと衝突した場合は、段数に関係なく無条件に倒せます。

ぱにっ娘がエイリアンに捕まるか、`OXYGEN` がなくなると、ぱにっ娘が1人死にます。

### 応用テクニック

#### 連鎖落とし

上の敵を落とすと、下の敵も連鎖で落下します。  
この場合、段数に関係なく無条件に倒せます。

```text
    ぱ
  H□敵□□□
  H
  H□　□□□
  H
  H□敵□□□
  H
  H□　□□H
  H　　　H
□□□□□□□□
```

#### 衝突

上の敵を落として、下の敵と衝突させます。  
この場合も、段数に関係なく無条件に倒せます。

```text
    ぱ
  H□敵□□□
  H
  H□　□□□
  H 敵→
  H□□□□□
  H
  H□□□□H
  H 　　H
□□□□□□□□
```

#### 埋め殺し

下が塞がっている場合、敵を埋めてしまいます。  
この場合も、段数に関係なく無条件に倒せます。

```text
    ぱ
    □敵□□□□
    □□□□□□
```

## 5. キー操作

### ゲーム中

```text
                           上
                           8
X, トリガA : 穴を掘る     左 4   6 右
Z, トリガB : 穴を埋める        2
                           下
```

- **X / トリガA**：穴を掘る
- **Z / トリガB**：穴を埋める
- **2 / 4 / 6 / 8**：移動

**※ `JOYSTICK` にも対応しています。**  
**※ `2 / 4 / 6 / 8` はカーソルキーにも対応しています。**

### PAUSE

ゲーム中に **ESCキー** を押す、またはジョイスティックで **A・Bボタンを同時押し** すると、ポーズになり `CONFIG` 設定に入ります。

メニュー画面では、`2` と `8`（またはジョイスティックの上下）で選択し、`Z`（またはジョイスティックのBボタン）で決定します。

#### メニュー画面説明

| 項目 | 内容 |
|---|---|
| `RETURN` | ゲーム継続 |
| `DEAD` | 強制的にぱにっ娘を1人殺す |
| `QUIT` | DOSに戻る |
| `4 SPEED 6` | スピード設定。数が少ないほど高速。`0`〜`99`、初期値 `10` |
| `4 SOUND 6` | 音のON（4キー）、音のOFF（6キー） |
| `4 ROUND 6` | 面セレクト（`001`〜`999`）。ただし現在は **32面まで** |

## 7. 設定ファイル（`PANIC.DEF`）について

このファイルを書き換えることで、各設定を変更できます。

| 項目 | 範囲 | 初期値 | 備考 |
|---|---|---|---|
| ぱにっ娘の数 | `01-09` | `03` |  |
| スピード | `00-99` | `10` | 値が大きいほど遅い |
| 開始面指定 | `001-099` | `001` |  |
| 不死身 | `00-01` | `00` | `01` で不死身 |
| BEEP | `00-01` | `01` | `00` で無音 |
| 面の最大値 | `001-099` | `32` |  |

**※ 半角文字で作成してください。**

---

## 8. マップファイル（`VMAP1.DAT` ～ `VMAP16.DAT`）

このファイルを書き換えることで、マップを変更できます。  
マップは **13×12ブロック固定** です。

### 例

```text
        A           A                      [意味]
  11H11111111H11
    H        H                    A ... SLIMER
  11H111111111111
    H  B   C                      B ... EBYRINS
  11H111111111111
    H                             C ... ZAUROID
  11H1111H111H1
    H   H     H                   ア ... SLIMER（落下）
  11H1111H111H1
    H M  H    H                   イ ... EBYRINS（落下）
  22222222222222
　　　　　　　　　　　　　　             ウ ... ZAUROID（落下）

　　　　　　　　　　　　　　　　　　　　　　  M ... ぱにっ娘
                                   1 ... ブロック
                                   2 ... 床
                                   7 ... 穴
                                スペース ... 道
                                          H ... はしご
```

**※ 半角文字で作成してください。**

### 制約事項

- 敵の数：`0～16`
