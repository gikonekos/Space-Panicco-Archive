# Space Panicco Archive（スペースぱにっこ アーカイブ）

1994年のDOSアクションゲーム **スペースぱにっこ**（BUGSOFT製）のアーカイブおよびドキュメントプロジェクトです。

---

## VMAP エディタ

Space Panicco の VMAPファイル（`.DAT`）用、ブラウザで動作するマップエディタです。

**ツール公開先:** https://gikonekos.github.io/Space-Panicco-Archive/tools/map-editor/

### 機能

- **13 × 12 グリッドエディタ** — マウスドラッグまたはタッチでセルを描画
- **内蔵32面**（ROUND 001〜032）— 実機データを収録
- **ROUND 001〜999 対応** — 数値入力フィールドで直接移動
- **読み込み / 保存** — CP932エンコードの `.DAT` ファイルをバイナリ読み書き（DOS EOF 0x1A 付き）
- **コピー** — 任意の面を別スロットに複製
- **Undo / Redo** — 50手履歴（Ctrl+Z / Ctrl+Y）
- **キーボードショートカット** — `SPC` `1` `2` `7` `H` `M` `A` `B` `C` `Q` `W` `E`
- **タッチ操作対応** — タブレット・スマートフォンでのドラッグ描画
- **MAP INFO パネル** — 全記号の配置数をリアルタイム表示

### 警告バナー

マップの問題を自動検出し、ツールバー直下に警告を表示します。

| 警告 | 条件 |
|---|---|
| PANICCO (M) 未配置 | M がマップに存在しない |
| PANICCO (M) 重複 | M が2個以上ある |
| 敵数超過 | 敵の合計が16体を超えている |
| 飛ばし番号警告 | 現在面より前に空面が存在する（ゲームは `not open - VMAPxxx.DAT !!` を表示して空面で続行）|
| フリーズリスク | 通常敵（A/B/C）の真下・隣接列に着地可能な床がない — スポーン直後にフリーズ |
| 袋小路フリーズ *(v1.5)* | 着地は可能だが、着地後の行で左右いずれかに連続2マス以上の通行可能スペースがない — 着地後にフリーズする可能性あり |

### 記号一覧

| 記号 | キー | 説明 |
|---|---|---|
| (SP) | Space | 道（敵が歩く空間） |
| `1` | 1 | ブロック（壁） |
| `2` | 2 | 床（プラットフォーム） |
| `7` | 7 | 穴 |
| `H` | H | はしご |
| `M` | M | ぱにっ娘（プレイヤー開始位置） |
| `A` | A | SLIMER（敵） |
| `B` | B | EBYRINS（敵） |
| `C` | C | ZAUROID（敵） |
| `ｱ` | Q | SLIMER — 落下スポーン |
| `ｲ` | W | EBYRINS — 落下スポーン |
| `ｳ` | E | ZAUROID — 落下スポーン |

### ゲーム挙動の確認済み仕様

- 空面でもクラッシュしない（詰みになるだけで続行）。
- ぱにっ娘（M）はマップ上になくても空中から出現する。
- 通常敵（A/B/C）は隣接1マス以内に床（着地可能セル）が必要。なければスポーン直後にフリーズ。
- **袋小路フリーズ（v1.5）:** 着地に成功しても、着地行で左右いずれかに連続2マス以上の空きスペースがない場合、敵は着地後にフリーズする可能性がある。エディタはこのパターンを検出して警告する。
- 飛ばし番号が存在すると、該当面で `not open - VMAPxxx.DAT !!` を表示し空面で続行する。

### ファイル形式

各 `.DAT` ファイルはCP932エンコードの12行テキストファイルです。

```
<マップ文字13文字><余白><コメント>\r\n
...（12行）
\x1a  （DOS EOF）
```

半角カタカナ（`ｱ` `ｲ` `ｳ`）はCP932バイト値（0xB1 / 0xB2 / 0xB3）でそのまま書き出します。
