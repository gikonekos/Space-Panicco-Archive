# FreeDOS(98) ライセンスメモ

このファイルは、実行および動作確認の便宜のために同梱している PC-98 用 DOS 起動環境の起動画面で確認できた表示を整理したものです。

これは FreeDOS(98) や関連構成要素の元ライセンス文書そのものを置き換えるものではありません。
本リポジトリで使用している同梱環境の起動時に、画面上で確認できた内容を記録したメモです。

## リポジトリ向けメモ
